# ViChae (비채) - 비우고 채우는 가치 있는 거래 🌿
**프로젝트 시연 시나리오 및 핵심 기능 소개**

---

## 🛠 기술 스택 (Tech Stack)
- **Frontend**: Next.js (App Router), React 19, Tailwind CSS v4
- **Backend/BaaS**: Supabase (PostgreSQL, Auth, Realtime, Storage)
- **Styling**: 현대적이고 다이내믹한 UI (Glassmorphism, Gradient)

---

## 🎬 시연 순서 (Demo Flow)

### 1. 플랫폼 첫인상 및 메인 페이지 (Home)
![메인 페이지 시연](C:/Users/user/.gemini/antigravity-ide/brain/eeb823b6-e517-40c9-a366-6a461a7a11e9/demo_home_1782805733641.webp)
- **접속 및 UI 확인**: 감각적이고 부드러운 애니메이션이 적용된 히어로(Hero) 섹션을 보여줍니다.
- **최신 상품 목록**: 최근 등록된 상품들이 그리드(Grid) 형태로 깔끔하게 렌더링되는 모습을 확인합니다.
- **맞춤형 피드**: 차단된 유저(Block)의 상품은 메인 피드에 노출되지 않도록 필터링되는 로직을 설명합니다.

### 2. 회원가입 및 로그인 (Authentication)
![회원가입 및 로그인 시연](C:/Users/user/.gemini/antigravity-ide/brain/eeb823b6-e517-40c9-a366-6a461a7a11e9/demo_auth_1782805998628.webp)
- **로그인 페이지 진입**: 우측 상단 '로그인' 버튼을 통해 인증 페이지로 이동합니다.
- **Supabase Auth**: 안전하고 빠른 이메일/소셜 로그인 과정을 시연합니다.

### 3. 내 물건 비우기 - 상품 등록 (Sell)
![보호된 라우트 및 회원가입 시연](C:/Users/user/.gemini/antigravity-ide/brain/eeb823b6-e517-40c9-a366-6a461a7a11e9/demo_sell_1782806045067.webp)
- **보호된 라우트**: 로그인하지 않고 `/sell`에 접근 시 자동으로 로그인 페이지로 리다이렉트 됩니다.
- **상품 등록 페이지 (`/sell`)**: 메인 화면의 '내 물건 비우기' 또는 네비게이션을 통해 이동합니다.
- **이미지 업로드 및 상세 정보 입력**: 상품 사진을 업로드하고, 제목, 가격, 설명을 입력하여 새 상품을 등록합니다.
- **상품 상세 확인**: 등록이 완료된 후, 상세 페이지로 이동하여 정보가 정상적으로 노출되는지 확인합니다.
> [!NOTE]
> *현재 데모 영상은 이메일 인증이 필요하여 상품 등록 폼 진입 전 화면(회원가입 및 보호된 라우트 접근)까지 시연되었습니다.*

### 4. 상품 탐색 및 상호작용 (Products & Like)
- **전체 상품 둘러보기 (`/products`)**: 카테고리나 최신순으로 다른 유저의 상품을 탐색합니다.
- **좋아요(찜) 기능**: 마음에 드는 상품에 하트 버튼(LikeButton)을 클릭합니다.
- **실시간 알림 (Real-time Notifications)**: 상품 판매자 측 화면(혹은 다른 브라우저)에서 Supabase Realtime을 통해 즉각적으로 알림(RealtimeNotifBadge)이 도착하는 것을 보여줍니다.

### 5. 실시간 채팅 및 거래 진행 (Real-time Chat)
- **채팅 시작**: 구매 희망자가 상품 상세 페이지에서 '채팅하기'를 클릭합니다.
- **1:1 실시간 채팅 (`/chat/[roomId]`)**: 
  - 메시지를 입력하면 상대방에게 새로고침 없이 즉각 전달됩니다.
  - 상단 헤더에 현재 거래 중인 **상품 요약 정보**가 표시됩니다.
- **거래 상태 변경**: 판매자는 채팅방 상단에서 상품 상태를 '예약중' 혹은 '거래완료'로 즉시 변경할 수 있습니다 (Transaction Controls).
- **채팅 뱃지**: 네비게이션 바에 실시간으로 안 읽은 채팅 개수가 뱃지(RealtimeChatBadge)로 표시되는 것을 확인합니다.

### 6. 마이페이지 및 프로필 관리 (My Page)
- **프로필 수정 (`/profile`)**: 닉네임 변경 및 프로필 사진 업데이트 기능(ProfileEditor)을 시연합니다.
- **나의 활동 내역**: 내가 등록한 상품, 판매 완료된 상품, 찜한 상품들의 목록을 마이페이지에서 한눈에 확인합니다.

---

## 💡 시연 시 강조 포인트 (Key Highlights)
1. **실시간 경험 (Real-time Experience)**: 채팅과 알림이 새로고침 없이 즉각적으로 반영되는 점(Supabase Realtime 활용).
2. **세련된 UI/UX**: Tailwind CSS v4를 활용한 둥글고 부드러운 컴포넌트, 다이내믹한 인터랙션과 반응형 디자인.
3. **사용자 보호 기능**: 원치 않는 사용자를 차단하여 피드와 채팅에서 제외시키는 안전한 거래 환경 제공.
